_# ANICEF 프로젝트

최종본

ANICEF 프로젝트 
<img src='WebContent/images/main.png'>

최근 유기 동물이 많아져 사회문제로 여겨지고 있습니다. 

대부분의 사람들이 유기 동물을 입양하지 않는 이유로 꼽히는 것이 
깨끗하지 않을 것이라는 선입견과, 보호 센터의 접근성이라고 
생각합니다.

 관심 있는 모든 사람들이 자유롭게 열람하고 좀 더 쉽게
 접근할 수 있도록 도와주는 사이트입니다.
